# 项目简介
## 核心功能
- 前后端代码及SQL文件
- 万字文档
- 导入调试视频、讲解视频及开发工具安装包

## 适用场景
- 需要构建机票预定系统的企业或个人
- 开发者需要参考的源码及文档
- 学习SpringBoot和Vue技术的开发者

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
